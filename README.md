# MURZA

Личный финансовый трекер — накопления, задачи, баланс.

Приложение работает прямо в браузере как PWA (Progressive Web App) — устанавливается на телефон без App Store и работает офлайн.

---

## Установка на телефон

### iPhone (Safari)
1. Открой сайт в **Safari**
2. Нажми кнопку **Поделиться** (квадрат со стрелкой вверх)
3. Выбери **«На экран "Домой"»**
4. Нажми **Добавить** — готово

### Android (Chrome)
1. Открой сайт в **Chrome**
2. Нажми **⋮** (три точки) в правом верхнем углу
3. Выбери **«Добавить на главный экран»** или **«Установить приложение»**
4. Подтверди — приложение появится на рабочем столе

---

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub (например `murza`)
2. Загрузи все файлы в корень репозитория
3. Зайди в **Settings → Pages**
4. В разделе **Source** выбери ветку `main`, папку `/ (root)`
5. Нажми **Save**
6. Через минуту сайт будет доступен по адресу: `https://твой-логин.github.io/murza`

## Возможности

- **Накопления** — цели с прогресс-баром, эмодзи-иконки
- **Задания** — задачи с вознаграждением в рублях, категории
- **Дашборд** — общий баланс, статистика, быстрые действия
- **Офлайн** — работает без интернета после первого посещения
- **Тёмная тема** — изумрудно-чёрная палитра
